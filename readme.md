## Installation
#### Install Dependencies
- ```npm install```
- Rename and configure mysql_config.example.js to mysql_config.js

#### Run Tests
- ```./node_modules/mocha/bin/mocha```

#### Scrape and load DB
- ```node main.js```
- Help: ``` node main.js --help```
